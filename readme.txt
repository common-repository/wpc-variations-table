WPC Variations Table will replace dropdown selects with a beautiful table.

== Description ==

Table layout for products has always been a fav for WooCommerce sites. The brand new **WPC Variations Table for WooCommerce** plugin is a perfect choice for your use. It replaces the drop-down for variations of variable products and coats your products with a nice, stunning table layout.

What **WPC Variations Table** brings to your site:

- Offer a charming, neat presentation of variations
- Enhance the user experience & shopping flow
- An easier way to introduce large variable products

= Live demo =

Visit our [live demo](https://demo.wpclever.net/woovt/ "live demo") here to see how this plugin works.

= Features =

- Setting an active status for all variable products across your store
- Show/hide out-of-stock/unpurchasable variations from the list of choices
- Scrolling vertically/horizontally when the number of variations is large
- Define variation name format â€“ default, with/without attribute labels
- Show/hide variation details: price, thumbnail, stock, description
- Separate quantity selectors per variations
- Separate Add to Cart buttons per variations
- Compatible with other WPC plugins: [WPC AJAX Add to Cart](https://wordpress.org/plugins/wpc-ajax-add-to-cart/) & [WPC Added to Cart Notification](https://wordpress.org/plugins/woo-added-to-cart-notification/)
- Customize variation table at product basis (Premium version only)

Wanna save your precious time working on variations? Try our brand-new free plugin [WPC Variation Bulk Editor](https://wordpress.org/plugins/wpc-variation-bulk-editor/) and [WPC Variation Duplicator](https://wordpress.org/plugins/wpc-variation-duplicator/).

= Perfect match with WPC plugins =

When using with [WPC AJAX Add to Cart](https://wordpress.org/plugins/wpc-ajax-add-to-cart/) and [WPC Added to Cart Notification](https://wordpress.org/plugins/woo-added-to-cart-notification/) plugins, WPC Variations Table can expose its maximum strength. **AJAX Add to Cart** helps deliver the chosen variation to the cart fast and properly without the need to reload the whole page to see the changes.

Plus, **WPC Added to Cart Notification** notifies customers about which have been added and prompt them with the next actions: View Cart or Continue Shopping. These complete the shopping circle for variable products on your site.

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC Variations Table", Install & Activate it
4. Go to WPClever > Variations Table to change settings
